# Tasha Jordan

واجهة موقع/تطبيق ويب لحجز العقارات داخل الأردن، مبنية باستخدام HTML وCSS وJavaScript.

## الصفحات
- index.html
- properties.html
- property.html
- bookings.html
- payment.html
- owner-dashboard.html
- admin-dashboard.html
- login.html
- register.html
- api-reference.html

## الميزات
- تصميم RTL مناسب للعربية مع خيار تبديل اللغة من الشريط العلوي
- تبديل العملة بين JOD وUSD
- مساعد اقتراحات ذكي محلي داخل JavaScript
- حجز مع خيار دفع كامل أو عربون
- تنزيل سند الحجز بصيغة PDF
- PWA basic (manifest + service worker)
- لوحات للمالك والإدارة

## التشغيل
افتح index.html مباشرة، أو ارفع المشروع إلى GitHub Pages أو Vercel.
